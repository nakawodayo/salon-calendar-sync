## セットアップ手順

### 1. 環境設定（ユーザー作業）

詳細は [USER_SETUP_GUIDE.md](./USER_SETUP_GUIDE.md) を参照してください。

**必要な作業**:
1. Firebase プロジェクトの作成
2. LIFF アプリの作成
3. Google OAuth 2.0 クライアントの作成
4. `.env.local` ファイルの作成

### 2. プロジェクトの起動

```bash
cd frontend
npm run dev
```

### 3. LIFF ブラウザで開く

LINE アプリで以下の URL を開く:
```
https://liff.line.me/<your-liff-id>
```
