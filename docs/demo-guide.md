# サロン予約アプリ デモガイド

## このアプリについて

LINEからお客さまが予約リクエストを送り、美容師さんがPCやスマホで確認・承認すると、Googleカレンダーに自動で予定が登録されるアプリです。

## 用意するもの

- **スマホ** — LINEが入っているもの（お客さま役で使います）
- **PC またはタブレット** — ブラウザが使えるもの（美容師側の操作で使います）
- **Googleアカウント** — Googleカレンダーと連携するために必要です

## Step 1: Googleアカウントを連携する（美容師側・PC）

1. PCのブラウザで以下のURLを開きます
   - **https://salon-calendar-sync.vercel.app/stylist/auth**
2. 「Googleアカウントでログイン」ボタンを押します
3. Googleのログイン画面が表示されるので、普段使っているGoogleアカウントでログインします
4. 「許可」を押してカレンダーへのアクセスを許可します
5. 「認証が完了しました」と表示されれば成功です

> ここで連携したGoogleアカウントのカレンダーに、予約が自動で登録されるようになります。

## Step 2: お客さま役で予約リクエストを送る（スマホ）

1. スマホのLINEで、以下のリンクを開きます
   - **https://liff.line.me/2008484018-nMrvLmk6**
2. ホーム画面が表示されるので「予約リクエストを作成」を押します
3. 以下の情報を入力します
   - **お名前** — テスト用なので適当でOKです
   - **メニュー** — カット、カラーなど選択します
   - **希望日時** — 第1希望と第2希望を選びます
   - **備考** — 必要に応じて入力します（空欄でもOK）
4. 「送信する」を押します
5. 「予約リクエストを送信しました」と表示されれば成功です

## Step 3: 予約リクエストを確認する（美容師側・PC）

1. PCのブラウザで以下のURLを開きます
   - **https://salon-calendar-sync.vercel.app/stylist/requests**
2. 先ほどお客さま役で送った予約リクエストが一覧に表示されます
3. リクエストを押すと詳細画面が開きます

## Step 4: 予約を承認してカレンダーに登録する

1. リクエストの詳細画面で内容を確認します
2. 「承認する」ボタンを押します
3. 「予約を承認しました」と表示されれば成功です
4. この時点で、Googleカレンダーにも予約が自動で登録されています

## Step 5: Googleカレンダーを確認する

1. Googleカレンダー（https://calendar.google.com）を開きます
2. 承認した予約の日時にイベントが入っていることを確認できます
3. イベントにはお客さまの名前やメニューなどの情報が記載されています

## 却下する場合

リクエストの詳細画面で「却下する」ボタンを押すと、その予約リクエストを却下できます。却下した場合、Googleカレンダーには登録されません。

## 困ったときは

| こんなとき | 対処法 |
|---|---|
| Google認証の画面でエラーが出る | ブラウザのCookieを削除してからもう一度試してみてください |
| 予約リクエストが一覧に表示されない | ページを再読み込み（更新）してみてください |
| 承認したのにカレンダーに表示されない | Googleカレンダーのページを再読み込みしてみてください。数分かかる場合があります |
| LINEでアプリが開けない | LINEアプリを最新版に更新してみてください |
| それでも解決しない場合 | 下記の連絡先にお知らせください |
